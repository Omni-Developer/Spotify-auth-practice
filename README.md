# Mini Spotify Clone

A full-stack Spotify-inspired music streaming application built with **React** and **Express.js**. The project features music playback, REST API integration, database connectivity, and dynamic routing, providing a seamless client-server experience.

## Tech Stack

- React
- Express.js
- Node.js
- Database (replace with MongoDB/MySQL)
- REST API

## Installation

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```
